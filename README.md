# FitBot - Your AI Fitness Coach

FitBot is an intelligent fitness coaching application designed to help users achieve their health goals through personalized guidance. Built with the MERN stack, it offers a seamless and interactive experience for tracking fitness progress and getting expert advice.

## üöÄ Features

- **Personalized Coaching**: Get tailored fitness advice based on your goals.
- **Real-time Chat**: Interact with the AI coach in real-time for instant feedback.
- **Secure Authentication**: User accounts are protected with secure signup and login processes.
- **Session Management**: maintain your chat history and context across sessions.
- **Responsive Design**: optimized for both desktop and mobile devices.

## üì¶ Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/Shubham20040627/Chat-Bot.git
   cd Chat-Bot
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Environment Variables**
   Create a `.env` file in the root directory and add your credentials:
   ```env
   MONGODB_URI=your_mongodb_connection_string
   JWT_SECRET=your_jwt_secret
   PORT=3000
   ```

4. **Run the Application**
   ```bash
   npm start
   ```
   The app will run at `http://localhost:3000`.
